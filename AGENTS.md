# AI Agents Documentation

## Overview

This project uses AI-powered recommendation features.

## Components

### Mood Analysis
Analyzes the user's selected mood and recommends suitable songs.

### Recommendation Engine
Provides:
- Playlist suggestions
- Related moods
- Recommended genres

## AI Providers

### Local AI
- Ollama
- User-configurable endpoint
- User-configurable model

### Cloud AI
- OpenAI (BYOK)
- Google Gemini (BYOK)

## Configuration

Users can configure AI providers through the Settings panel.
