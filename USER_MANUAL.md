# User Manual

## Overview

Mood-Based Music Recommender is an AI-powered application that suggests songs based on user mood.

## Features

- Mood-based recommendations
- Multi-language support
  - English
  - Hindi
  - Telugu
- Local AI using Ollama
- BYOK support for OpenAI and Gemini

## How to Use

1. Open the application.
2. Select your preferred language.
3. Choose a mood.
4. Click "Get Recommendations".
5. View recommended songs and playlist suggestions.

## Settings

Use Settings to:
- Change language
- Select AI provider
- Configure Ollama
- Enter OpenAI API Key
- Enter Gemini API Key
